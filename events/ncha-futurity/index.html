About NCHA Futurity

The NCHA Futurity is one of the premier cutting events in the world, governed by NCHA and serving as a major proving ground for top open competitors. For western horse fantasy fans, this event represents a future expansion opportunity for the Fantasy Run For A Million platform.

Fantasy coverage of this event is planned as the platform expands beyond The Run For A Million. When activated, this page will include eligible rider rosters, fantasy scoring configuration, and real-time leaderboard tracking.
